## 1-2 실습 과제 

클래스 기본 실습

- Counter 클래스 만들기
  - 프로퍼티
    - number (Int), 초기값 0
  - 메서드: addCount라는 이름으로 number 값에 1을 더하고 number를 return


- ViewController 만들기 (스토리보드)
  - 프로퍼티

    - counter (Counter)
  - 메서드
    -   executeCounter 라는 이름으로 countButton 누를시 counter의 addCount를 실행하고 label에 표시
